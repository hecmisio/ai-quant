# AI量化-常见技术指标

## 概览

常见技术指标并不只是“会背公式”，更重要的是理解它们分别在回答什么问题：

- `K线` 在看多空博弈过程
- `成交量` 在看行情有没有资金推动
- `市值` 在看标的体量和弹性
- `PE / PB` 在看估值是否偏贵或偏便宜
- `ROE` 在看公司盈利能力
- `自由现金流` 在看利润质量和真实造血能力
- `MA / MACD` 在看趋势方向和趋势速度
- `RSI` 在看短期情绪是否过热或过冷
- `ATR` 在看波动和风控尺度
- `KDJ / WR` 在看震荡区间里的短线情绪位置
- `OBV` 在看资金流向是否支持趋势
- `布林带` 在看价格相对波动区间的位置

## K线图

K线图可以理解为多空双方的一场战报，核心由四个价格和两类形态组成。

### 参数

- 开盘价（Open）
- 收盘价（Close）
- 最高价（High）
- 最低价（Low）

### 实体（Body）

- 阳线：`收盘价 > 开盘价`，说明买方占优。实体越长，买方越强。
- 阴线：`收盘价 < 开盘价`，说明卖方占优。实体越长，卖方越狠。

### 影线（Shadow）

- 上影线：买方冲高后回落，通常表示抛压较大，上方有人想逃跑。
- 下影线：卖方杀跌后被拉回，通常表示支撑较强，下方有人在抄底。

## 成交量

成交量是行情的燃料。一个常见比喻是：价格像车，成交量像油。

### 常见观察方式

- 量价配合：车往上爬坡时，通常需要踩油门。价格上涨往往需要放量配合。
- 缩量上涨：通常意味着动力不足，容易滞涨，很多时候更偏向诱多信号。
- 量价背离：价格创新高，但成交量没跟上，往往是见顶预警信号之一。

## 市值

市值 = 股价 × 总股本

市值主要帮助我们理解一家公司“有多大”，以及它的价格行为通常会更像哪一类资产。

### 常见分类

- 大盘股：如银行、两桶油，通常像大象，走势更稳，但跑得慢。
- 小盘股：如偏题材股，通常像猴子，跳得高，但也更容易摔死。

## 市盈率（PE）

市盈率（PE）可以粗略理解为“回本需要多少年”。

- 常见计算方式：`市值 / 净利润`
- 或等价理解为：`股价 / 每股收益（EPS）`

### 如何理解

- 一般来说，PE 越低越好，说明估值相对更便宜。
- 但如果是科技股、AI 股这类市场愿意为未来高增长买单的公司，PE 可能会很高，比如 `50` 倍、`100` 倍甚至更高。
- 实操里，很多人会先过滤掉 `PE > 100` 的股票，除非它属于成长性极强的优质公司。

## 市净率（PB）

市净率（PB）可以理解为“如果公司现在清算，资产还值多少钱”。

- 常见计算方式：`股价 / 每股净资产`

### 如何理解

- `PB = 1`：意味着市场大致按“1 块钱资产卖 1 块钱”来定价。
- `PB < 1`：常被理解为“破净”，意味着公司像在打折卖，银行、钢铁等行业更常见。

### 使用提醒

只看 PB 低不一定是真的便宜。如果一家公司长期亏损，净资产可能会不断缩水。量化选股时，PB 往往要结合 `ROE` 一起看。只有那些还能持续赚钱的“破净股”，才更有可能是真正被低估。

## 净资产收益率（ROE）

ROE 是衡量公司盈利能力的核心指标之一。

- 常见计算方式：`净利润 / 股东权益（净资产）`

### 如何理解

- 优秀公司：如果能长期保持 `15% ~ 20%` 以上的 ROE，尤其是持续 `5 ~ 10` 年，通常说明公司拥有较强护城河。
- 及格线：很多投资框架会认为，`ROE < 8%` 的公司回报率跑不赢通胀和机会成本，投资价值偏低。

## 自由现金流

自由现金流常用来判断企业的真实造血能力。

- 可关注每股自由现金流
- 常见理解方式：`经营性现金流 - 资本开支`

### 自由现金流 vs PE

净利润可能会因为折旧、摊销等会计规则被修饰，但现金流往往更接近真实经营情况。也就是说，利润表可能“好看”，但现金流才更像真金白银。

一个典型理解方式是：

- 公司短期利润不好看，甚至亏损，不一定代表经营差。
- 如果它只是因为扩张而投入了大量设备、产能或基础设施，导致账面利润被压低，但经营性现金流依然很强，那么说明它的造血能力可能依旧很强，未来潜力也可能更大。

## 均线（MA）

均线（MA, Moving Average）可以理解为市场的成本线。

- 直观理解：过去 `N` 天市场参与者的平均买入价格。
- 深度理解：均线本质上是一种低通滤波器，用来过滤高频噪音，只保留更低频、更平滑的趋势信号。

### 常见用法

- 支撑 / 压力：如果股价回落到 `MA20` 附近后不跌了，通常说明多头在护盘，这里可能形成支撑。
- 金叉 / 死叉：短周期均线如 `MA5` 上穿长周期均线如 `MA20`，通常说明短期力量强于长期力量，趋势可能启动；反过来则偏空。

### 如何理解 MA

- MA 看方向，是价格的平滑处理，主要告诉你当前更接近涨势还是跌势。
- 它的优点是稳，过滤噪音能力强，在像茅台这类大趋势行情中，往往能吃到最肥的一段。
- 它的缺点是滞后，很多时候行情已经走出来一截才发出信号。
- 在震荡市里，均线系统容易频繁来回打脸，也就是常说的“来回止损”。

## 指数平滑异同移动平均线（MACD）

MACD 可以理解为“速度的度量衡”，很多人也把它叫作趋势动量指标之王。

- 直观理解：看快慢线的交叉和红绿柱变化。
- 深度理解：MACD 计算的是短期均线和长期均线之间的距离变化，本质上在衡量价格上涨或下跌的加速度。

### 常见观察方式

- 红柱子变长：说明上涨正在加速，可以理解为“油门踩到底”。
- 红柱子变短：即使价格还在涨，也说明上涨速度慢下来了，属于一种背离或钝化前兆，是偏危险的信号。

### 如何理解 MACD

- MACD 看速度，也就是看动量，是均线系统的衍生物。
- 它比 MA 更敏感，反应更快，常被用来通过顶背离、底背离提前预判趋势转折。
- 但在单边暴涨暴跌行情里，MACD 也可能出现钝化，指标参考意义会下降。

## 相对强弱指标（RSI）

RSI 本质上是买卖双方力量对比的量化结果，也可以理解为上涨速度和下跌速度之间的比较。

### 常见观察方式

- `RSI > 80`：通常视为超买。连续大涨后，买方力量很强，但也意味着价格可能被推到极限，均值回归概率增大。
- `RSI < 20`：通常视为超卖。连续大跌后，卖方力量很强，但也意味着价格可能被压到极限，反弹概率增大。

### 如何理解 RSI

- RSI 更适合捕捉短线和震荡市场中的情绪拐点。
- 它常被用来寻找极端情绪背景下的反转信号。

## 随机指标（KDJ）

KDJ 是一种典型的震荡类指标，用来观察价格在近期区间中的相对位置，以及短线情绪是否过热或过冷。

### 如何理解

- 可以把 KDJ 理解为“股价现在处在最近一段区间的高位还是低位”。
- 当指标快速上冲时，说明短线买盘很强；当指标快速下探时，说明短线卖压很重。
- KDJ 对短线波动非常敏感，因此在震荡市里常比均线更好用。

### 常见观察方式

- 金叉：`K` 线上穿 `D` 线，通常视为短线转强信号。
- 死叉：`K` 线下穿 `D` 线，通常视为短线转弱信号。
- 高位钝化：指标长期停留在高位，说明行情很强，但也要提防冲高回落。
- 低位钝化：指标长期停留在低位，说明行情很弱，但也可能临近超跌反弹。

### 使用提醒

- KDJ 很灵敏，所以优点是反应快，缺点是假信号也多。
- 在单边趋势行情里，KDJ 容易频繁提示“超买”或“超卖”，但价格未必马上反转，因此更适合配合趋势指标一起看。

## 威廉指标（WR）

WR（Williams %R）和 KDJ、RSI 类似，也是一种衡量超买超卖的震荡类指标。

### 如何理解

- WR 用来衡量当前收盘价在最近一段时间最高价和最低价区间中的位置。
- 指标越靠近极端区间，说明情绪越容易过热或过冷。

### 常见观察方式

- WR 进入超买区：通常意味着价格短期涨得过快，后续容易出现震荡或回撤。
- WR 进入超卖区：通常意味着价格短期跌得过快，后续容易出现技术性反弹。

### 使用提醒

- WR 和 RSI、KDJ 很像，都更适合震荡市而不是单边趋势市。
- 如果趋势非常强，WR 长时间停留在极端区间并不罕见，所以不能只凭一个超买超卖信号就逆势交易。

## ATR（平均真实波幅）

ATR 是量化交易里非常重要的波动率指标，也是仓位管理和海龟交易法则中的核心工具之一。

### 如何理解

- ATR 不直接告诉你涨还是跌，而是告诉你“最近波动到底大不大”。
- 波动越大，说明风险越高，止损空间和仓位通常也要随之调整。
- ATR 常用于设置止损距离、跟踪止损和动态仓位控制。

## 能量潮指标（OBV）

OBV（On-Balance Volume）是一种把成交量和价格方向结合起来观察资金流向的指标。

### 如何理解

- 如果价格上涨时成交量也在不断累积，OBV 往往同步走高，说明资金在跟随流入。
- 如果价格上涨但 OBV 没有同步创新高，就可能意味着上涨缺少量能支持。

### 常见观察方式

- OBV 上升：说明买盘资金更积极，趋势更容易被确认。
- OBV 下降：说明卖盘压力更大，走势可能偏弱。
- OBV 与价格背离：价格创新高但 OBV 没创新高，或价格创新低但 OBV 没创新低，通常值得警惕趋势可能减弱。

### 使用提醒

- OBV 本身不负责给出精确买卖点，它更像一个“量价关系确认器”。
- 实操中常把它和均线、MACD 一起使用，用来判断趋势背后有没有资金配合。

## 布林带（Bollinger Bands）

布林带是一种把“均线”和“波动率”结合起来的指标，经常用来判断价格是否偏离正常波动区间。

### 如何理解

- 中轨通常是一条均线，可以理解为价格的中枢。
- 上轨和下轨则反映当前波动率背景下的正常波动边界。
- 价格靠近上轨，通常说明走势偏强；价格靠近下轨，通常说明走势偏弱。

### 常见观察方式

- 布林带开口变大：说明波动正在放大，趋势可能加速。
- 布林带收口变窄：说明波动正在收缩，后续可能酝酿方向选择。
- 价格反复触碰上轨：说明趋势很强，但也要防止短期过热。
- 价格跌到下轨附近：可能表示短线偏弱，也可能出现均值回归机会。

### 使用提醒

- 布林带既可用于趋势观察，也可用于震荡回归判断。
- 在强趋势中，价格“贴着上轨走”并不一定代表马上要跌；在震荡市中，上下轨的参考意义通常更强。

## 指标类型速览

不同技术指标更适合不同市场环境，把它们分组理解会更容易上手。

### 趋势型

- `MA`
- `MACD`
- 适用场景：单边牛市、单边熊市、明显趋势行情

### 震荡型

- `RSI`
- `KDJ`
- `WR`
- 适用场景：震荡市、区间市、均值回归环境

### 能量型

- `OBV`
- `成交量`
- 适用场景：趋势确认、量价验证

### 波动型

- `ATR`
- `布林带`
- 适用场景：风险控制、止损设置、波动率判断

## 每个指标适合什么行情

指标没有“万能答案”，关键是把指标放到合适的市场环境中使用。

| 指标 | 更适合的行情 | 不太适合的行情 | 核心原因 |
| --- | --- | --- | --- |
| K线 | 几乎所有行情，尤其适合做价格行为观察 | 单独使用时不适合做完整决策 | K线能提供最原始的多空信息，但缺少统计过滤 |
| 成交量 | 放量突破、趋势确认、情绪转折 | 冷门股、流动性差的标的 | 量能对趋势真伪很关键，但在失真成交里意义会下降 |
| 市值 | 选股分层、风格研究、容量评估 | 短线择时 | 市值更偏资产画像，不是短线买卖信号 |
| PE | 稳定盈利行业、成熟公司估值比较 | 周期顶部、亏损股、高成长题材股 | 利润波动大时，PE 容易失真 |
| PB | 银行、地产、钢铁等重资产行业 | 轻资产、高研发、无形资产占比高的行业 | PB 更适合看资产底，不擅长评估创新溢价 |
| ROE | 质量因子选股、长期价值筛选 | 一次性利润、财务杠杆过高的公司 | ROE 能看盈利效率，但可能被杠杆美化 |
| 自由现金流 | 价值投资、盈利质量判断 | 重扩张早期公司、强周期资本开支高峰期 | 现金流更真实，但短期会被扩张节奏扰动 |
| MA | 明显趋势行情、波段行情 | 高频震荡、无序横盘 | 均线擅长过滤噪音，不擅长抓震荡拐点 |
| MACD | 趋势启动、趋势加速、趋势衰减 | 极端单边加速后的钝化行情、低波动横盘 | MACD 擅长看动量，不擅长长期盘整 |
| RSI | 震荡市、超买超卖判断、短线反转 | 强单边趋势市 | 趋势太强时，超买超卖会失灵很久 |
| KDJ | 短线震荡、日内波动、区间交易 | 大级别趋势持仓 | KDJ 太敏感，适合短，不适合稳 |
| WR | 区间震荡、超跌超涨识别 | 趋势连续推进阶段 | 和 KDJ、RSI 类似，极端值不代表立刻反转 |
| ATR | 风控、止损、仓位管理 | 单独做方向判断 | ATR 衡量波动，不衡量方向 |
| OBV | 趋势确认、量价验证、背离观察 | 无量品种、被消息强拉强砸的异常阶段 | OBV 的前提是量能具有参考意义 |
| 布林带 | 震荡回归、波动收缩突破 | 单独判断大趋势方向 | 布林带擅长看区间和波动，不擅长单独定方向 |

## 常见误区

很多新手不是不会看指标，而是把指标放错了场景，或者赋予了它超出能力边界的含义。

| 指标 | 常见误区 | 更合理的理解 |
| --- | --- | --- |
| K线 | 看到一根大阳线就直接追 | 单根 K 线意义有限，要结合位置、成交量和前后结构 |
| 成交量 | 只要放量就是好事 | 放量也可能是出货，关键看价格是否同步走强 |
| 市值 | 小盘股一定涨得快，大盘股一定没弹性 | 市值影响风格和弹性，但不是绝对收益保证 |
| PE | PE 越低越好 | 低 PE 可能是价值，也可能是陷阱，先看利润是否可持续 |
| PB | PB 小于 1 就是便宜 | 破净不等于低估，烂资产也会一直便宜 |
| ROE | ROE 高就一定是好公司 | 高 ROE 可能来自高杠杆、回购或一次性利润 |
| 自由现金流 | 现金流差就是公司不行 | 扩张期公司可能短期现金流承压，要结合阶段判断 |
| MA | 金叉就是必涨，死叉就是必跌 | 均线信号本质上是滞后确认，不是预言 |
| MACD | MACD 金叉一出现就重仓冲 | MACD 更适合辅助确认趋势节奏，不适合脱离结构单独重仓 |
| RSI | RSI 超买就必须卖，超卖就必须买 | 超买超卖说明极端，不代表拐点马上发生 |
| KDJ | KDJ 一钝化就一定见顶或见底 | 强趋势里钝化可以持续很久 |
| WR | WR 到极值就立刻反手 | WR 适合提示情绪极端，不适合孤立决策 |
| ATR | ATR 上升就是要跌 | ATR 上升只说明波动变大，不区分上涨还是下跌 |
| OBV | OBV 上升就一定有主力进场 | 它反映量价关系，不等于能准确识别资金身份 |
| 布林带 | 碰上轨必跌，碰下轨必涨 | 强趋势里价格可能沿着轨道持续运行 |

## 组合使用建议

单个指标只能回答一个角度的问题，更实用的方式是把“方向、动量、情绪、量能、风控”拼成一个完整决策链。

### 1. 趋势跟随组合

- 适用行情：单边上涨、单边下跌、突破后延续行情
- 常见组合：`MA + MACD + OBV + ATR`
- 使用思路：
  - 用 `MA` 判断大方向是否顺势
  - 用 `MACD` 看趋势是否在加速或衰减
  - 用 `OBV` 确认量能是否支持趋势
  - 用 `ATR` 设止损和仓位，避免趋势对了但仓位扛不住

### 2. 震荡回归组合

- 适用行情：横盘整理、区间震荡、无明显单边趋势
- 常见组合：`RSI + KDJ + WR + 布林带`
- 使用思路：
  - 用 `布林带` 判断价格是否靠近区间边界
  - 用 `RSI / KDJ / WR` 识别超买超卖和情绪拐点
  - 多个震荡指标同时给出极端信号时，反转参考价值更高

### 3. 突破确认组合

- 适用行情：平台突破、箱体突破、收缩后放量选择方向
- 常见组合：`成交量 + MA + 布林带 + MACD`
- 使用思路：
  - 先看 `布林带` 是否收口，确认市场在压缩波动
  - 再看价格是否站上关键 `MA`
  - 用 `成交量` 判断突破是否真实
  - 用 `MACD` 判断突破后是否有加速迹象

### 4. 价值筛选组合

- 适用行情：中长期选股、基本面筛选、低估值研究
- 常见组合：`PE + PB + ROE + 自由现金流 + 市值`
- 使用思路：
  - 用 `PE / PB` 看估值是否便宜
  - 用 `ROE` 看公司是否具备持续赚钱能力
  - 用 `自由现金流` 验证利润质量
  - 用 `市值` 判断标的容量、波动特征和风格属性

### 5. 从 0 到 1 的简化决策框架

可以把大部分指标按下面的顺序理解：

1. 先问方向对不对：看 `MA`
2. 再问趋势强不强：看 `MACD`
3. 再问位置是不是太极端：看 `RSI / KDJ / WR / 布林带`
4. 再问有没有量能支持：看 `成交量 / OBV`
5. 最后问风险自己扛不扛得住：看 `ATR`

如果是做基本面和中长期配置，则可以换成：

1. 先看贵不贵：`PE / PB`
2. 再看赚不赚钱：`ROE`
3. 再看利润是不是真实：`自由现金流`
4. 最后看适不适合自己的风格：`市值`

## 新手使用建议

- 不要一次学十几个指标，先用 `MA + 成交量 + ATR` 建立最基础的趋势和风控框架。
- 趋势指标和震荡指标不要混着硬用。趋势市优先看 `MA / MACD`，震荡市优先看 `RSI / KDJ / WR / 布林带`。
- 指标是辅助你做决策，不是替你思考。位置、结构、量能、风控通常比单个数值更重要。
- 一个指标给信号，不如一组逻辑给结论。越是重要的交易决策，越应该有多个维度互相验证。
- 先决定自己是做短线、波段还是中长期，再决定看哪些指标。周期不一致，信号就会打架。
